Abstract  
This  paper  presents  the  design,  implementation  and  validation  of  an  automated  Security  
Information
 
and
 
Event
 
Management
 
(SIEM)
 
system,
 
orchestrated
 
through
 
n8n
 
and
 
built

with
 
open-source
 
technologies.
 
The

to
 
centralize
 
logs,
 
detect
 
anomalies
 
and
 
execute
 
automatic
 
responses
 
to
 
incidents
 
in

controlled

reducing
 
mean

to
 
detect
 
and
 
respond
 
(MTTA/MTTR)

for

security
 
operations

distance

The  solution  integrates  syslog-ng,  Logstash,  Elasticsearch,  PostgreSQL,  Grafana  and  n8n,  
forming

Results

reduced
 
triage

greater
 
visibility
 
into
 
security
 
events,
 
full
 
playbook

functional

brute-force
 
attack
 
detection,
 
and
 
file

monitoring
 
(FIM)
 
via
 
specialized
 
Python
 
scripts,
 
with
 
a
 
measured
 
automation

of
 
94.64%.
 
Keywords:  SIEM,  n8n,  SOAR,  cybersecurity,  automation,  Grafana,  PostgreSQL,  open  
source,
